# YouTube Style Fixer

This is a simple Firefox extension that improves the visual design of YouTube by injecting custom CSS and JavaScript.

---

## How to Build and Install

### 1. Requirements
- Operating System Windows, macOS, or Linux  
- Browser Firefox 109 or later  
- Build tools Not required (no Node.js, npm, or transpilation needed)

---

### 2. Source Files
All source code is written in plain JavaScript and CSS.  
No minification, bundling, or compilation is used.

File structure
├── manifest.json
├── content.js
├── styles.css
├── icon.png
├── README.md
└── build.sh
